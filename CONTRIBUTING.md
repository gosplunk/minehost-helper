# Contributing

MineHost Helper is source-available, but it is not open source. Please read [LICENSE.md](LICENSE.md) before contributing.

Useful contributions right now:

- Bug reports with clear steps to reproduce.
- Screenshots or logs with public IPs, private IPs, usernames, webhook URLs, and passwords removed.
- Documentation fixes.
- Small pull requests that the maintainer has already agreed to review.

Do not submit large rewrites, rebrands, installer changes, or bundled third-party binaries without opening an issue first.

By submitting code for inclusion, you grant the project owner the contribution rights described in [LICENSE.md](LICENSE.md).

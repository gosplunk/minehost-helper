Install MineHost Helper

Follow these steps exactly. You only need two clicks after extracting the ZIP: trust the friend publisher certificate, then run the installer.

1

Extract the ZIP

Right click MineHostHelper-FriendSigned.zip and choose Extract All. Do not run files from inside the ZIP preview.

2

Run the certificate helper

Double click Install MineHost Helper Friend Publisher Certificate.bat. Approve the Windows administrator prompt.

3

Run the installer

Double click MineHostHelperSetup-FriendSigned.exe. Follow the installer and let it open the browser.

Only approve the certificate if you trust the friend who sent this

The certificate tells Windows to trust MineHost Helper friend builds from that person. If Microsoft Defender says the file is a virus or blocks it with WinError 225, stop and ask your friend. Do not disable Defender.

What file should I click?

  1. First: Install MineHost Helper Friend Publisher Certificate.bat
  2. Second: MineHostHelperSetup-FriendSigned.exe
  3. Do not click Source code downloads from GitHub.

After install

  1. Create your MineHost Helper web login.
  2. Choose Setup new server, guided.
  3. Use the Dashboard buttons to start, stop, and copy the friend address.

Login page

The browser opens to the local MineHost Helper login.

Choose guided setup

Pick Setup new server, guided unless you already have a Minecraft server folder.

Use the Dashboard

Start or stop the server and copy the address for friends from the Dashboard.
